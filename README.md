# Utility Tracker — Property Consumption Dashboard

A modern full-stack dashboard for tracking utility **readings** and **consumption trends** across multiple properties.  
Built with **Next.js**, **PostgreSQL**, **Drizzle ORM**, **Tailwind CSS**, and **shadcn/ui** — fully typed, fast, and ready for deployment on **Vercel**.

**This project is a demo, feel free to use it for studying purposes**

---

## Features

- **Dashboard Overview**
  - Consumption totals
  - Last 30 days chart
  - Recent readings table

- **Readings Management**
  - List all readings with property names
  - Clean, dark, minimalistic UI
  - Placeholder UI when no data is present

- **Property Support**
  - Multiple properties
  - Readings associated with each property
  - Easy extension to add rates, billing, alerts, etc.

- **Dark Theme**
  - Fully styled with Tailwind
  - Reusable shadcn/ui components
  - Sidebar layout inspired by Vercel dashboard

- **Database Layer Using Drizzle**
  - Strong typing
  - Safe migrations
